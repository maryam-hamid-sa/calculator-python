# Python Calculator

A simple calculator made with Python that performs basic arithmetic operations.
<img width="597" height="335" alt="images (1)" src="https://github.com/user-attachments/assets/47040f65-f82e-4bca-b228-c71d53ff83bd" />



## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Handles division by zero error

## Requirements

- Python 3.x

## How to Run

1. Download or clone the project.
2. Open terminal or command prompt.
3. Run the following command:
   python calculator.py
4. Enter two numbers and choose an operator when prompted.

## Example
Enter first number: 26

Choose operator (+, -, *, /): -

Enter second number: 29

Result: -3.0
