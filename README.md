# Python Calculator

A simple calculator made with Python that performs basic arithmetic operations.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Handles division by zero error

## Requirements

- Python 3.x

## How to Run

1. Download or clone the project.
2. Open terminal or command prompt.
3. Run the following command:
   python calculator.py
4. Enter two numbers and choose an operator when prompted.

## Setup and Installation

To run the calculator locally, follow these steps:

1. Clone the repository:
   
   git clone https://github.com/maryam-hamid-sa/calculator-python.git

3. Navigate to the project directory:
   
   cd calculator-python

5. Run the calculator:

   python calculator-python.py

7. Follow the on-screen prompts to enter numbers and choose an operator.

## Example
Enter first number: 26

Choose operator (+, -, *, /): -

Enter second number: 29

Result: -3.0
