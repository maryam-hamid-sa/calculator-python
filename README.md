# Python Calculator

A simple calculator made with Python that performs basic arithmetic operations.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Handles division by zero error

## Requirements

- Python 3.x

## How to Run

1. Download or clone the project.
2. Open terminal or command prompt.
3. Run the following command:
   python calculator.py
4. Enter two numbers and choose an operator when prompted.

## Example
Enter first number: 26

Choose operator (+, -, *, /): -

Enter second number: 29

Result: -3.0
